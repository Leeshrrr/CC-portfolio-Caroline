Project #4 Exquisite Corpse

Description

This is an exquisite corpse composed by Brandon’s, Charlotte’s and mine corpse. I use mine’s head, Charlotte’s body and Brandon’s legs to make the final work.

My inspiration of the corpse is from the famous work ”the Scream”, Charlotte’s is a random scrawl man, and Brandon’s is a man with a long neck. And I call my exquisite corpse “the scrawl”.

The sketch from Charlotte, Brandon and me, and also the final corpse

Design Process

Code sourse link here

This is my basic structure. I created functions to draw all parts of the corpse and use parameter to control the color and weight of the strokes.

To express the strokes, I used a library called rough.js. It can help us to draw graphics in a style of sketch. We can input basic graphics like rectangles or paths in svg.

Reflection

Function is good at it's reusability. If we need to repeat a method for many times or will frequently changed the parameter, using function is a GREAT idea.
