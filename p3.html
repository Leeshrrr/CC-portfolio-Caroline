Project #3 Experimental Clock

Description

Do you want to use the Time Turner

press the mouse: change the hair style

mouseX: change the skin color and eye movement

mouseY: change the background color, eyemovement and eyebrow movement

press any key: save your portrait to download

Design Process

Code sourse link here

1. Change skin color and background color: build a function to make color changes between color 1 and 2 [colorScale(mouseX or mouseY, color1, color2)]. Method utilized: switch-case; constrain; map;

2. Switch hair style randomly: Method utilized: mousePressed; random; randomSeed; switch-case; loadImage;

3. Move eyes and eyebrows: build functions [drawEyes(Xposition, Yposition)]. Method utilized: 2D primitives; mouseX & mouseY;

4. Save image: Method utilized: keyPressed; DOM;
