Project #5 Optical Illusion

Drag your mouse.

Description

There is a traditional handicraft in my country called blueprint. People print repetitive patterns on the cloth like this:

And I tried to imitate the pattern in a digital way.

As we should apply Optical Illusion, I construct the “blueprint” using a famous pattern and modify it to be the basic element:

It seems like a flat pattern initially, but if we drag the canvas by mouse, the sight changes and it becomes a 3 dimensional object. In some specific angles, the object will construct other patterns.

Here are some tries.

Design Process

Code sourse link here

As to make the pattern changeable I use WEBGL so I can rotate the objects. I use a library called EasyCam.js to make the angle of sight change by dragging the mouse.

As the repetition of boxes and basic patterns is regular, so I use for loop to construct it.

Reflection

“What are some impacts and effects of translating the pattern of your choice into digital, animating form? How does creating an optical illusion recontexualizes how the pattern was traditionally used & understood?”

I’m not quite sure. Actually we can’t make the optical illusion whatever we want. There should be a “coincidence” to make the construction of optical illusion available. So it is difficult for me to combine my cultural tradition with it… And many optical illusions are just based on the animation. So do there must be some recontexualization? I don’t think so.
