Project #5 Optical Illusion

Hover your mouse on the spots to see the place name.

Description

I choose my footprint from Oct 1 to Oct 7 as the data. The map will show my movement from place to place, connecting them by lines, and you can check the place I visit by hover the mouse on the white points. You can also zoom in and out the map or change the position just like the way in Google Map.

Design Process

Code sourse link here

I use “timeline” in Google Map to track my way and download the JSON file. The data includes the position, the name of place, the duration etc. Then I use loadJSON to fetch the data and also put the position in to arrays.

Then I use Mappa.js to apply the map. I use leaflet as the source of map and use the function latLngToPixel to map the latitude and longitude data to points on the canvas.

And I use lerp() and millis() to make dynamic lines.

Reflection

Data visualization can restore physical data like the position or duration, but the subjective things like emotions and thoughts can not be shown in it. (Of course I can tell it by myself, but it can not be collected by apps etc.) So the things we can do further is to add our feelings into it.
